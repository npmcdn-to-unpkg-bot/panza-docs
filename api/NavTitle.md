## NavTitle 
 
Renders text stylized for the Navigation Bar title.

On iOS it composes PrimaryText.

On Android, it composes LargeText.

 
 __Composes__: [Text](Text.md) 


### Props
Name | Type | Default Value | Description
--- | --- | --- | --- 
label | string  |   | 
color | string  |   | 
style | any  |   | 
inverted | bool  |   |
